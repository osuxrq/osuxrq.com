菜鸡杰克的初中阶切指练习推荐（进阶已补完）

#1446CN#15Jack_Wang_98.73% Lv.102(61%) 13253PP

正文

大概的难度标准：

  1. 继续努力
  2. 又是一个新人群扛把子，走了
  3. 可以提了.jpg
  4. 您诗人？？

注意：星数并不一定代表某一张图的实际难度。请熟悉用按照物件摆放、排列、数量等来衡量难度的方法。

Stage 1 笨鸟先飞

4.6321946NEW AstronomasMemme // Charles445[Color's Another]osu!standard

作为入门熟悉数量杂乱切指还是很有帮助的。OD 不高，acc 不会太难打，注意HP即可。

难度：1

要求：98accfc

5154727fake me (Science Adventure Dance Remix)Itou Kanako // ykcarrot[Insane]osu!standard

主打短切以及移动。注意OD以及HP。

难度：1-2

要求：95acc pass -> 任意acc s rank -> 98accfc

51296762Zauberkugelxi // pishifat[Insane]osu!standard

对于 aim 要求较高，切指不会是太大难点。CS 方面稍微留意就没有问题。 难度 1-2-2.6 要求：不低于94acc pass -> 95accfc--高于97acc

4.9763294Dark Flight DreamerALiCE'S EMOTiON // Sakaue Nachi[Twaoi's Insane]osu!standard

不要被bpm吓到，整张图打起来十分舒服，可以作为前期基础练习移动和切指的保留图。

难度：1-2

要求：95acc以上pass -> 99acc以上fc

4.8568565Chinese Restaurant中国餐馆Memme // Muya[Hyper]osu!standard

排列和切指数量都比较清晰的一张图，主要注意的是切分手指一定要清楚，要不然很容易少切或者多切一个。Kiai中的滑条接二连需要注意一下。

难度：1-2

要求：95acc pass -> 98accfc

4.8420012snow storm -euphoria-雪暴Yooh // ktgster[EXHAUST]osu!standard

奇数切练习，没有过多的aim难点。也可以拿ar8洗洗眼睛（笑

难度：1

要求：97acc以上fc

Stage Clear!

Stage 2 切指初心者

4.8964065Cherry BlossomMemme // Priti[Karen's Insane]osu!standard

对于刚刚练习切指一段时间的人来说，这张图很好的考验了综合切指+移动能力，fc可能不是什么难事，但是acc就不好说了。

难度：1.5-2

要求：95acc pass -> 97acc以上fc

5.11242786Oriental BlossomEisyo-kobu // Crystal[EmingK's Insane]osu!standard

低OD，排列可能一开始稍微有点卡手，注意不要过多次重试，反而会出梗，等到对自己实力有一定信心以后可以试试高 acc。

难度：1.5-2.5

要求：97accfc -> SS

4.9306038Kobito of the Shining Needle ~ Little Princess小公主ZUN // sjoy[Lunatic]osu!standard

中低速切指，作为入门练习还是很好玩的。注意后面的短串和衔接上可能出现的问题，出现16note连打时不要慌，抬高手指控住即可。

难度：2-2.5

要求：95acc以上pass -> 97acc以上fc

5.3667282%S(M)Memme // P o M u T a[AngelHoney's Insane]osu!standard

手控自测图，排列和切指数量极为综合，同时od不是很高，acc也要注意一下

难度：2-2.5

要求：90acc pass -> 95acc pass -> 97acc以上fc

4.8965494crystallizedCamellia // Smoothie World[wa's Insane]osu!standard

主要是较低的AR带来的读图+炫酷排列，注意拉滑条的方式，不然acc会很难拉。同时要留意在连打+短切中的aim。

难度：2-2.5

要求：93acc pass--不低于95acc fc -> 97acc以上fc

4.81295717AvalancheMemme // Starfy[Insane]osu!standard

大量的滑条+短切+衔接，较低的bpm也会是一个问题，手控要求较高。滑条也将会是一个acc的大难点，注意拉滑条的时机。

难度：2.5-3

要求：不低于95acc pass -> 97acc以上fc

Final Boss

5743818Dark Flight DreamerALiCE'S EMOTiON // Natsu[Lunatic]osu!standard

物件摆放较为整齐，多四方+五角星+三角跳，同时注意不要梗掉。切指方面，主打3连和4连接滑条，无太多难点，注意BPM可能对手控带来的影响。

难度：2.5

过关要求：不低于95acc fc

51053845Warp DriveTitancube // DTM9 Nowa[Another]osu!standard

低 BPM 和较高的 OD 带来的手控问题将会是一大难点，多偶数切和滑条短串的衔接，移动要求中等。

难度：2.5

过关要求：不低于96acc fc

Stage Clear!

Stage 3 初阶学员

5.1386140Orange Filedeadmau5 // Jethh[Extra]osu!standard

acc可能是个问题，手控要注意。

难度：2-2.5

要求:不低于97acc fc -> SS

5.1786021Night of Knights夜骑beatMARIO // alacat[iyasine's Lunatic]osu!standard

中规中矩的一张切指图，奇数偶数切都有，打起来也比较顺手。稍稍注意一下od对acc可能出现的影响。

难度：2-3

要求：不低于93acc pass -> 96acc fc -> 99acc fc

5917430True Blue真蓝dj TAKA feat.AiMEE // ZZHBOY[Insane]osu!standard

难度稍有提升，低bpm和乍一听起来非常不合拍的碎切都是cb和acc的阻碍。切记要把手指抬清楚。

难度：2.5-3

要求：93acc pass -> 96acc以上fc

5.3899859Flashes (Extended)Dollscythe // handsome[Atsuro's Insane]osu!standard

非常考验综合能力的一张长图，180 BPM 下的od7.5对于acc的影响全在于你自己的手控。也算是中间的一个小小检查点吧（笑

难度：2.5-3

要求：不低于95acc pass -> 97acc以上fc

5715345Starving DaysMemme // Gamu[Another]osu!standard

难度较高，高 BPM 带来的切指压力会变得越来越大，不断的三连和 4、5 连衔接更是 acc 杀手。

难度2.5-3.5

要求：不低于95acc fc--不低于98acc fc

5.21009793polygonSota Fujimori // Sebu[tetragon]osu!standard

继续进阶，kiai 段的滑条和衔接是一个难点，基础碎切没什么过于困难的地方。od 低，但是 acc 不太好打。

要求：97acc -> 99acc

5.1818272Ryoushi no Umi no LindwurmKuroneko Dungeon // P o M u T a[SUPER]osu!standard

低速切指自测，注意排列对 aim 带来的可能影响。

要求：a rank pass--不低于96acc

5.1545555.357 Magnumxi // Akali[Hyper]osu!standard

排列以及切指数量都非常舒服的一张中度长度图，注意手控。

要求：不低于95acc pass -> 97acc fc--把我从榜上爆掉

Checkpoint

5.1589965AiAeYuyoyuppe // Fort[Another]osu!standard

很考验稳定性的一张图，acc方面要注意。

5.11427700CallingZekk // NeilPerry[Mirash's Another]osu!standard

继续 ar8 洗眼睛（笑

其实也可以试试 hr，但是 none 都切不到 a 评级的话就暂时别打了。

5.1571591Wish upon Twin Starsxi // Chaoslitz[EXHAUST]osu!standard

主要是 BPM 带来的切指问题，od影响不会太大。

要求：97acc 以上 fc

5.3557821RenatusSoleily // Multiple Creators[Insane]osu!standard

萌新必备曲（大雾），od 和 bpm 搭配可能会出现一些问题，不过不会太大。

要求：97acc 以上 fc
